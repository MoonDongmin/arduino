### 이용하는 센서: 조도센서, 초음파센서, LED, 부저
### 작동 방법: 밝기가 어두울 때 센서가 작동함. 어두울 때는 초음파 센서를 이용하여 앞에 있는 물체를 감지함. 초음파의 거리에 따라 LED의 밝기 달라지며. 사용자와 물체와의 거리가 가까우면 부저가 울리며 위험하다고 알려주는 방식
